1. Overview

meshTerm is an SSH terminal client for iPhone and iPad. It lets you connect to remote servers using password or SSH key authentication, and integrates with Tailscale to browse and connect to your Tailnet devices directly from the app.

The main host list dashboard.

2. Prerequisites

For all connections

  • A server with SSH enabled and accessible from your device
  • Your server's hostname or IP address, port, and a valid username

For SSH key authentication Pro

  • An Ed25519 SSH key pair (Generate in-app or import unencrypted OpenSSH key)
  • The public key added to ~/.ssh/authorized_keys on the server

3. Getting Started

Adding your first host

  1. Open meshTerm and tap + in the top-right corner.
  2. Fill in the Label, Hostname, Port, and Username.
  3. Under Authentication, choose your method and credentials.
  4. Tap Add to save.
Adding a new SSH host.
Editing existing host details.

4. Managing SSH Hosts

Swipe right on a host and tap Edit, or swipe left to Delete. Free accounts are limited to 1 saved host, while Pro removes this limit.

6. Authentication Methods

Password: Stored securely in the iOS Keychain.

SSH Key Pro: Uses Ed25519 keys; the private key never leaves the device.

Tailscale SSH Pro: Authenticates via your Tailscale identity.

7. SSH Key Management

Access keys via Settings → SSH Keys. You can generate new Ed25519 keys or import existing unencrypted OpenSSH private keys.

Key management and import options.

8. Tailscale Integration

Integrate with Tailscale to browse your Tailnet and add devices directly as SSH hosts. This requires a read-only API token entered in Settings → Tailscale.

Browsing active devices on your Tailnet.

9. Snippet Vault Pro

A searchable library of terminal commands that can be sent to an active session with one tap. Includes default folders for Linux and OpenClaw admin commands.

The Snippet Vault command library.

10. Terminal Controls

Active shell with modifier row.
The extended symbol strip.
Function (F1-F12) and navigation keys.

The terminal includes one-shot modifiers for Ctrl, Esc, and Alt, plus a dedicated Tab key for autocompletion.

11. Security

meshTerm features Trust On First Use (TOFU) for host key verification and App Lock (Face ID/Touch ID/Passcode) to protect the application.

14. Troubleshooting

If you encounter Local Network connection issues, ensure the permission is enabled in iOS Settings → Privacy & Security → Local Network. For host key mismatches, delete the old fingerprint in Settings → Known Hosts.
