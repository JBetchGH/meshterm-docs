meshTerm User Guide

meshTerm is a powerful SSH terminal client specifically designed for iPhone and iPad. It bridges the gap between mobile productivity and remote server management with native Tailscale integration.

Prerequisites

For all connections

  • SSH-enabled server reachable via network.
  • Hostname/IP, Port (default 22), and Username.

For Pro Features Pro

  • Tailscale App: Must be installed and connected for Tailscale SSH and device browsing.
  • Ed25519 Keys: Required for SSH key authentication.

Getting Started

Adding a host is the first step to managing your infrastructure from your pocket.

  1. Tap + in the top-right corner.
  2. Enter your Label, Hostname, and Username.
  3. Choose your Authentication Method.
  4. Tap Add and then tap the host in your list to connect.

Managing SSH Hosts

Control your list with simple gestures:

  • Edit: Swipe right or long-press on a host.
  • Delete: Swipe left and tap Delete.
  • Limits: Free accounts are limited to 1 host. Upgrade to Pro for unlimited hosts.

Authentication Methods

Password

Stored securely in the iOS Keychain. It is never written to disk in plain text.

SSH Key Pro

Uses local signing. Your private key never leaves the device's Secure Enclave.

Tailscale SSH Pro

Tailscale SSH authentication is handled entirely by the Tailscale app — meshTerm does not send any credentials for these connections.

SSH Key Management Pro

Manage your keys in Settings → SSH Keys.

Generating Keys

meshTerm generates modern Ed25519 key pairs. Tap Generate to create one and copy the public key to your server's ~/.ssh/authorized_keys.

Importing Keys

Paste your unencrypted OpenSSH private key. Note that meshTerm does not support passphrases for keys currently.

ssh-keygen -p -N "" -f ~/.ssh/id_ed25519

Tailscale Integration Pro

Browse your entire Tailnet by tapping the globe icon in the host list.

API Token Required: You must generate a read-only API token in your Tailscale admin console to enable device browsing.
  1. Generate token at tailscale.com.
  2. Set permissions to read-only.
  3. Paste into Settings → Tailscale → API Token.

Snippet Vault Pro

Stop typing complex commands on a mobile keyboard. Access your vault via the snippets button in the terminal toolbar.

  • Linux Essentials: Common admin commands included.
  • OpenClaw: Specialized commands for OpenClaw users.
  • Searchable: Quickly find commands by title or tags.

Terminal Controls

Key Action
CtrlOne-shot modifier for sequences like Ctrl+C.
EscExit insert mode or cancel prompts.
TabShell autocompletion.
AltMeta modifier for word-jumping (Alt+F/B).

Security

Trust On First Use (TOFU)

meshTerm verifies server identity via fingerprints. If a host key changes, meshTerm blocks the connection to prevent MITM attacks.

App Lock

Secure the app using Face ID, Touch ID, or Passcode via Settings → App Lock.

Troubleshooting

Host Key Mismatch

If you see a mismatch warning, only delete the entry from Settings → Known Hosts if you are 100% sure the server was rebuilt.

Tailscale Issues

  • Ensure the Tailscale app is connected.
  • Verify "Tailscale SSH" is enabled on the destination server (sudo tailscale up --ssh).
