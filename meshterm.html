User Guide

meshTerm is a native SSH terminal client for iPhone and iPad, allowing you to manage infrastructure on the move with professional tools and Tailscale integration.

The main host list dashboard.

Getting Started

To connect to a server, you first need to add it to your host list. meshTerm supports standard SSH parameters including custom ports and environment variables.

Adding a new SSH host.
Editing existing host details.

Environment Variables

Custom environment variables can be sent to the shell upon connection. Add them in the Environment Variables section of the host configuration.

Authentication Methods

  • Password: Stored securely in the iOS Keychain.
  • SSH Key (Pro): Ed25519 public key authentication.
  • Tailscale SSH (Pro): Secure identity-based login without managing credentials.

SSH Key Management Pro

meshTerm uses Ed25519 keys for maximum security. You can import existing keys using standard iOS text tools or the document scanner.

Importing keys using Scan Text or Paste.

To remove a passphrase from a key before importing, use:

ssh-keygen -p -N "" -f ~/.ssh/id_ed25519

Security

Known Hosts (TOFU)

meshTerm uses Trust On First Use to verify server identity. You will be prompted to accept a fingerprint on the first connection.

The security prompt for new host keys.

Troubleshooting

Connection Refused

Check that sshd is running on the server and that your device has Local Network permissions enabled in iOS Settings.

Authentication Failure

Ensure your public key is present in ~/.ssh/authorized_keys and that file permissions are set correctly (700 for .ssh, 600 for the keys file).
